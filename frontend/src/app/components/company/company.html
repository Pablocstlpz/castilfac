Sobre CastilFac

Transformando la gestion de los talleres de carpinteria metalica a traves de la tecnologia y la innovacion.

Nuestra Mision

CastilFac nace con un objetivo claro: digitalizar y optimizar el flujo de trabajo de los profesionales del aluminio y el cristal. Historicamente, la creacion de presupuestos y el despiece de materiales ha sido un proceso manual, lento y propenso a errores de calculo que cuestan dinero a los talleres.

Esta plataforma SaaS centraliza clientes, tarifas y calculos matematicos complejos en una interfaz limpia e intuitiva, permitiendo a los talleres centrarse en lo que mejor saben hacer: fabricar e instalar.

El Origen del Proyecto

Este software ha sido desarrollado integramente como Proyecto de Fin de Grado (TFG) para el Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (2DAM).

Arquitectura y Tecnologias

Frontend

Interfaz de usuario reactiva, rapida y modular (SPA), diseñada para ofrecer la mejor experiencia en cualquier dispositivo.

  • Angular 17+
  • Tailwind CSS
  • Angular Material

Backend & API

Servidor robusto que gestiona la logica de negocio, el calculo de presupuestos y la seguridad mediante tokens JWT y CORS.

  • Node.js
  • Express.js
  • JSON Web Tokens (JWT)

Base de Datos

Arquitectura Multi-Tenant estructurada para garantizar el aislamiento y la privacidad de los datos de cada taller registrado.

  • MySQL Relacional
  • Borrado Logico (Seguridad)
  • Encriptacion bcrypt
